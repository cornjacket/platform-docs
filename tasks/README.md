# Platform-Docs Task Documents

Project-wide tasks that affect workspace structure, cross-repo concerns, or project-level process. These are distinct from `platform-services/tasks/` which tracks service code changes.

## Scope

| Location | Scope |
|----------|-------|
| `platform-services/tasks/` | Code, tests, service-level docs |
| `platform-docs/tasks/` | Workspace structure, cross-repo concerns, project process |

## Conventions

- Sequential numbering: `NNN-description.md` (independent from platform-services numbering)
- Same task/spec templates as platform-services (see `platform-services/tasks/README.md`)
- Task doc must be created before implementing changes
