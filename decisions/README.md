# Architectural Decision Records (ADRs)

This directory contains the Architectural Decision Records (ADRs) for the Cornjacket Platform. ADRs are short technical documents that capture important architectural decisions made during the project. Each ADR describes a design choice, the context surrounding it, the alternatives considered, and the consequences.

The ADRs serve as a historical log of architectural decisions, providing context and rationale for future reference and for onboarding new team members.
